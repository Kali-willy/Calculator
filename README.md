## How to Use

### Mouse Input

- Click on number buttons (0-9) to input numbers
- Click on operation buttons (+, -, √ó, √∑) to perform calculations
- Use AC to clear all input
- Use DEL to delete the last digit
- Click = to see the result
- Use % for percentage calculations
- Use . for decimal numbers

### Keyboard Support

- Numbers: `0-9`
- Operations:
  - Addition: `+`
  - Subtraction: `-`
  - Multiplication: `*`
  - Division: `/`
- Equal: `Enter` or `=`
- Delete: `Backspace`
- Clear All: `Escape`
- Decimal Point: `.`
